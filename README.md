README - Discover Oman Website Project

This is a College Project for the University of Technology and Applied Sciences (UTAS), Suhar Branch, 
developed as part of the CSWD1103 Web Development Course.

Project Title: Discover Oman
Description:
The project showcases Omani culture and heritage through a website that includes sections on 
traditional clothing, cuisine, handicrafts, music, and more. It also features a simple 
Currency Exchange page built with HTML and JavaScript.

Notes:

  and may differ from real-world values.
